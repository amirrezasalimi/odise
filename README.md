# odise

full open source noteboollm alternative


### features
- Import: PDF, EPUB, TXT, DOCX, Excel, CSV, HTML, Markdown, Links
- providers: kokoro-tts, pocket-tts
- supported Items:
    - Audio book reader [working on it]
    - visualizer [soon]
    - chat [soon]
    - podcast [soon]
    - Infography [soon]
    - mindmap [soon]
    - quiz [soon]
    - research canvas [soon]
